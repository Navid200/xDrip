# Glucose Retention
[xDrip](../../) >> [Features](../Features_page.md) >> Glucose Retention  
  
This setting can be found under Settings &#8722;> Less common settings.  
  
By default, it is set to `0`, which means no data is ever deleted.  
If you set a non-zero value, xDrip will automatically delete data older than that number of days on a daily basis.
