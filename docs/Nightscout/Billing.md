# Google Cloud Billing
[Google Cloud Nightscout](./GoogleCloud.md) >> Billing  
  
Go to [Google Cloud](https://cloud.google.com/).  
Sign in.  
Click on "Console".  
![Console](./images/Console.png)  
  
Click on the main menu at the top left and select "Billing".
