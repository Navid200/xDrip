## Update Nightscout
[Google Cloud Nightscout](./GoogleCloud.md) >> Update Nightscout  
  
To check your current version of Nightscout, open Nightscout in a browser, click on the hamburger menu button, and scroll down to the "About" section. There, you will find the Nightscout version listed.  
![](./images/Check4Updates.png)  
  
To see if an update is available, visit the "Installation" section of the Google Cloud Nightscout website. The version mentioned in parentheses indicates the Nightscout version provided by a fresh installation. If your current version is older, you can update it using the steps outlined below.  
  
Please note that you must have already completed the initial Nightscout [installation](./NS_Install.md) steps before attempting an update.  
<br/>  
  
---  

#### **How to update**  
Follow these [steps](./NS_SyncExecutables.md) to update your virtual machine platform to the latest version.  
  
Once the process is complete, your Nightscout should be updated. If it isn't, this likely means that the platform has not yet been updated, and further testing is still underway. In such cases, refer to the section below.  
<br/>  

---  

#### **What if no update is available?**    
When a new version of Nightscout is released, it must first be installed on Google Cloud to verify its compatibility and functionality within our setup. During this process, we may need to adjust the Linux packages used to support the new release.  
Only after thorough testing and validation will we release an updated installation routine. At that point, the version listed on the Google Cloud Nightscout page will be updated to reflect the new release. You will then be able to use the standard update method mentioned earlier.  
  
This means that immediately after a new Nightscout version is released, there may be a brief period when you cannot update using the usual process. If this happens, we recommend being patient for a few days while we complete our verification and release.  
If you're unsure or need assistance, feel free to [inform us](./GCNS_Support.md).
