# Cannot enter a treatment
[xDrip](../../) >> [FAQ](../FAQ_page.md) >> Why can I not enter a treatment?  
  
If you enter a value in the time tab that is not in a valid 24-hour format, you will not be able to enter a treatment.  You will need to either delete the time value or correct it.
