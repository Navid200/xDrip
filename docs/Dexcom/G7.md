## G7, One+ or Stelo  
[xDrip](../../README.md) >> [Features](../Features_page.md) >> [xDrip & Dexcom](../Dexcom_page.md) >> Dexcom G7, One+ or Stelo  
  
This is how you can use xDrip to collect directly from G7, One+, or Stelo.  
<br/>  
  
---  
  
#### **Starting (inserting) the sensor**  
Before inserting a new sensor, use a pen to mark the old one to avoid accidentally removing the wrong sensor later.  
  
Inspect the sensor for any [damage](./G7ApplicatorFailures.md), then follow the manufacturer’s instructions to insert it using the applicator.  
The session will start automatically once the sensor is inserted—[no need to manually stop or start](./WhyNoG7Stop.md) anything.  
  
Make a note of the 4-digit pairing code displayed on the sensor applicator, as you will need it to establish connectivity later.  
![](./images/G7_Applicator.png)  
<br/>  
  
---  
  
#### **Establishing Connectivity**  
  
1- If you have the Dexcom app (or any other app collecting from the sensor) installed, uninstall it.  Do not start your Dexcom receiver or t:slim pump yet.  If they're already running, disable them and wait 15 minutes.  
  
2- Go to Android Settings &#8722;> Connected devices.  
Forget all devices named are DXCM\*\*,  DX01\*\* or DX02\*\*, where \*\* represents any two characters.  
  
3- Ensure xDrip is updated to at least the stable release version.  
[How to Update](../Updates.md)  
  
4- [Scan the QR code](../Settings/Scan_QR_Code.md) below if you haven't already.  This step is required only once and not for every sensor.  
![](./images/G7_keks_QR.png)  
  
You should see this message:  
![](./images/keks_QR_confirm.png)  
Approve importing the settings.  
<br/>  
  
5- Set up xDrip, and your phone, as explained [here](../G6-Recommended-Settings.md).  Be sure to review and confirm all settings, as the QR code doesn't configure them automatically.  
  
6- Ensure your old sensors are out of range or place them in a microwave.  If your old sensor is still working, it's okay as long as it's connected to another instance of xDrip. If you've disconnected it, peel it off and move it away.  
  
7-  Enter the 4-digit pairing code from the sensor applicator as the transmitter ID.  
![](./images/DexG7ID.png)  
<br/>  
  
8- Go to the [Dex status page](../StatusG5G6.md), and wait for a connection.  
If you're using Android 10 or newer, approve the pair request.  
  
If you see "Deep Sleeping errors", restarting the phone may help.  
<br/>  
  
If the sensor was inserted more than 30 minutes ago, you should see readings. If it was just inserted, it will show as "warming up."  Over time, the status page will populate with additional data, including time remaining and other elements.  
<br/>
