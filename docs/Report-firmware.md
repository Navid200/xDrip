## Report Firmware
[xDrip](../) >> [Features](./Features_page.md) >> [xDrip & Dexcom](./Dexcom_page.md) >> Report firmware  
  
If the firmware version shown on the system status page is unknown and should be reported, check the [release notes](./ReleaseNotes.md).  If you see a newer release updating the firmware, update to that version.  
If no such release is listed in the release notes, please [inform us](https://github.com/NightscoutFoundation/xDrip/discussions).
